## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ installed
- Google Gemini API key ([Get one here](https://makersuite.google.com/app/apikey)) - **FREE tier available!**

### Installation

1. **Clone the repository**

```bash
cd seva-mela
```

2. **Install dependencies**

```bash
npm install
```

3. **Set up environment variables**

Create a `.env.local` file in the root directory:

```bash
# Required: For quiz generation and background image generation
GEMINI_API_KEY=your_gemini_api_key_here
```

4. **Run the development server**

```bash
npm run dev
```

5. **Open your browser**

- On the same machine: Navigate to [http://localhost:3000](http://localhost:3000)
- From other devices on the network: Use your local IP address (e.g., `http://192.168.x.x:3000`)

  To find your local IP:

  - **macOS/Linux**: Run `ifconfig | grep "inet " | grep -v 127.0.0.1`
  - **Windows**: Run `ipconfig` and look for IPv4 Address

  The dev server will display all available network addresses when it starts
